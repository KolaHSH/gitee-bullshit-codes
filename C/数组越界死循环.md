- **数组越界死循环**

```
int main()
{
    int a[10], n;
    for (n = 0; n <= 10; n++) {
        a[n] = n - 10;
        printf("%d\n",a[n]);
    }
    printf("end\n");
    return 0;
}
```

​    

​    在内存分配上，按**a[0],a[1],a[2]…a[9],n** 排列 ,在for循环中有数组越界：a[10] = 10 - 10, a[10]对应的内存位置

正好是n的位置所在, 把n重新写成了0，所以一直死循环…^-^
